Health Radar v1.6.0 — Hybrid AI Assistant

Додано:
- кнопка 🤖 AI Асистент у шапці;
- окрема вкладка 🤖 AI Асистент;
- локальний AI-аналіз без API-ключа;
- режим ChatGPT API за бажанням;
- збереження OpenAI API key тільки локально у браузері;
- готові питання: тиск, погода, ліки, тригери, що показати лікарю;
- попередження, що AI не замінює лікаря.

Важливо:
- OpenAI API key не зберігати в GitHub.
- Якщо використовуєш ChatGPT API, ключ вставляється тільки у вікні AI Асистента.
- Локальний AI працює безкоштовно і без ключа.

Перевірка:
1. У шапці має бути v1.6.0.
2. Натиснути 🤖 AI Асистент.
3. У локальному режимі поставити питання.
4. Має зʼявитися відповідь без API-ключа.
