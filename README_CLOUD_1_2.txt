Health Radar Cloud 1.2

Додано:
- редагування записів погоди;
- видалення записів погоди;
- чекбокси для масового видалення тиску, ліків, подій, погоди;
- кращі кнопки редагування/видалення в журналах;
- автосинхронізація після редагування/видалення;
- перемикач автосинхронізації у блоці “Хмара”;
- виправлений Firebase apiKey.

Як оновити GitHub:
1. Розпакуйте архів.
2. На GitHub у health-radar натисніть Add file → Upload files.
3. Перетягніть усі файли з архіву із заміною існуючих.
4. Commit changes.
5. Через 1 хв натисніть Ctrl+F5 на сайті.
