Health Radar v1.6.3 — Cloud Sync Fix

Виправлено головну причину, чому на телефоні не видно даних з ПК:
- app.js тепер робить state глобальним через var state + window.state.
- firebase-cloud.js тепер завантажує дані з Firestore саме в основний state додатка, який використовує UI.
- після Google-входу, якщо локально на телефоні 0 записів, а у хмарі є записи — дані автоматично завантажуються з хмари.
- якщо і локально, і в хмарі є записи — виконується об’єднання.
- Service Worker оновлено до v1.6.3 і чистить старий кеш.

Після оновлення:
1. Відкрити сайт.
2. Ctrl+F5 на ПК.
3. На телефоні в Chrome: Налаштування сайту → Сховище → очистити дані для kyzja.github.io або відкрити в інкогніто.
4. Увійти Google тим самим акаунтом.
5. Дані повинні з’явитися автоматично.
